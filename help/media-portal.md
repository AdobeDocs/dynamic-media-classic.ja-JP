# Media Portal{#media-portal}

Adobe Mediumポータルでは、企業は承認されたアセットを容易に取得、制御し、外部のパートナーやチャネル、および内部ユーザーに配布できます。 ブラウザーベースの「セルフサービス」環境では、管理者が制御したビューをAdobe Dynamic Media Classic Assets に公開して、アセットを簡単にアップロード、参照、検索、プレビュー、会社が承認した形式で書き出すことができます。

必要システム構成など、Media Portal について詳しくは、 [Adobe Dynamic Media Classic Media Portal の使用](https://help.adobe.com/en_US/scene7/mediaportal/) <!-- (https://help.adobe.com/en_US/scene7/mediaportal/index.html) -->.

Media Portal の管理については、[クイックスタート：Media Portal の管理](quick-start-media-portal-administration.md#quick_start_media_portal_administration)を参照してください。
