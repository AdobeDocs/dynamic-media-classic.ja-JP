# アップロード時の画像微調整オプション{#image-editing-options-at-upload}

AI、EPS、PSD ファイルなどの画像ファイルをアップロードする際、アップロードオプションを設定ダイアログボックスで次の編集操作を実行できます。

* 画像の端から余白を切り抜きます。
* 手動での画像の端の切り抜き
* カラープロファイルの選択
* クリッピングパスからのマスクの作成
* アンシャープマスクオプションによる画像へのシャープの適用
* 背景のノックアウト

これらのオプションは、**[!UICONTROL 画像の編集オプション]**&#x200B;見出しの下にあるアップロードページにあります。

## 画像から空白を切り抜く

画像から空白のピクセルを自動的に切り抜くことができます。 アップロードジョブオプションダイアログボックスで、**[!UICONTROL 切り抜きオプション]**&#x200B;を選択します。 「**[!UICONTROL 切り抜き]**」ドロップダウンリストで、「**[!UICONTROL トリミング]**」を選択します。 次のオプションを選択します。

* **[!UICONTROL トリミングの基準]**：このドロップダウンリストから、カラーまたは透明度に基づいて切り抜くかどうかを選択します。

   * **[!UICONTROL カラー]**: **[!UICONTROL カラー]** オプションを選択します。 次に、**[!UICONTROL Corner]** ドロップダウンリストから、切り抜くホワイトスペースのカラーを最もよく表すカラーを含む画像の角を選択します。

   * **[!UICONTROL 透明度]**: 「透明度」オプションを選択します。

* **[!UICONTROL 許容値]**: スライダーをドラッグして、0 ～ 1の許容値を指定します。

   * **色に基づくトリミング**：画像の隅で選択した色と完全に一致する場合にのみ、ピクセルを切り抜くには0を指定します。 設定値が 1 に近づくにつれ、許容される色の違いが大きくなります。

   * **透明度に基づくトリミング**：透明なピクセルの場合のみ切り抜きピクセルに0を指定します。1に近い数値を指定すると、透明度が高くなります。

## 画像の側面から手動で切り抜く

画像の端を手動で切り抜くには、切り抜きメニューを選択し、「手動」を選択します。 次に、画像の一部または全部の辺から切り抜くピクセル数を入力します。 画像から切り抜かれる長さは、画像ファイルの ppi（ピクセル/インチ）設定によって異なります。 例えば、画像に150 ppiが表示されているとします。 次に、「上」、「右」、「下」、「左」テキストボックスに「75」と入力します。 この時点で、両側が0.5 インチ切り抜かれます。

## カラープロファイルの選択

画像のカラースペースを選択するには、「カラープロファイル」オプションを選択します。

* **[!UICONTROL sRGBに変換]**: sRGBに変換（標準の赤緑色の青）。 sRGBは、Web ページに画像を表示する際に推奨されるカラースペースです。

* **[!UICONTROL 元のカラースペースを保持]**：元のカラースペースを保持します。

* **[!UICONTROL カスタム送信元]** > **[!UICONTROL 送信先]**: メニューを開いて、「変換元」と「変換先」カラースペースを選択できるようにします。 標準のPhotoshop カラースペースまたはAdobe Dynamic Media Classicにアップロードしたカラースペースを選択できます。

詳しくは、[ICC プロファイル](icc-profiles.md#icc_profiles)を参照してください。

## クリッピングパスからのマスクの作成

クリッピングパス情報に基づいて画像のマスクを作成するには、**[!UICONTROL クリッピングパスからマスクを作成]**&#x200B;を選択します。 このオプションは、画像編集アプリケーションで作成される際に、クリッピングパスが作成された画像に適用されます。

## アンシャープマスクを使用した画像のシャープ化

このフィルターを使用すると、最終的なダウンサンプル画像に対するシャープフィルター効果を微調整できます。 効果の強度、効果の半径（ピクセル単位）、無視されるコントラストのしきい値を制御できます。

このエフェクトは、Photoshopのアンシャープマスクフィルターと同じオプションを使用します。 その名称とは異なり、アンシャープマスクはシャープ適用フィルターの一種です。

「アンシャープマスク」で、目的のオプションを設定します。 次の表に、設定オプションについて示します。

| アンシャープマスクのオプション | 説明 |
| --- | --- |
| 適用量 | 適用量は、エッジ ピクセルに適用されるコントラストの量を制御します。<br><br>効果の強度と考えてください。 Dynamic Media ClassicとAdobe Photoshopのアンシャープマスクの量の値には違いがあります。 主な違いは、Photoshopの金額範囲が1%から500%であることです。 Adobe Dynamic Media Classicの値の範囲は0.0 ～ 5.0です。 Adobe Dynamic Media Classicの5.0の値は、Photoshopの500%に相当します。0.9の値は90%に相当します。 |
| 半径 | 効果の半径を制御します。 <br><br>値範囲は0 ～ 250です。 この効果は、画像内のすべてのピクセル上で実行され、すべてのピクセルから全方向に放射されます。 半径の単位はピクセルです。 例えば、2000 × 2000 ピクセルの画像と500 × 500 ピクセルの画像に同様のシャープ効果を適用するには、2000 × 2000 ピクセルの画像に2つのピクセルの半径を設定します。 次に、500 × 500 ピクセルの画像に1 ピクセルの半径値を設定します。 ピクセル数の多い画像ほど大きい値が使用されます。 |
| しきい値 | しきい値は、アンシャープマスクフィルターが適用されたときに無視されるコントラストの範囲です。 このエフェクトは、このフィルターを使用するときに画像に「ノイズ」が発生しないように重要です。 値の範囲は 0 ～ 255 です。これはグレースケール画像における明るさのステップ数です。 0=black、128=50% グレー、255=白。<br><br>例えば、しきい値12を指定すると、肌の色調の明るさが少し変化するので、ノイズが加わるのを防ぐことができますが、まつげと肌が合う場所など、コントラストの強い領域にはエッジ コントラストが加わります。<br><br>例えば、誰かの顔の写真がある場合、アンシャープマスクは画像の対照的な部分に影響を与えます。 例えば、まつげと肌が出会って明らかなコントラストが生まれる場所や、肌が滑らかになる場所などです。 滑らかな肌でも、明るさの値によって微妙な変化を示します。 しきい値を使用しない場合、このフィルターによって肌のピクセル内の微妙な変化が強調されます。 そのため、ノイズの多い望ましくない効果となる一方、まつげのコントラストが増加し、シャープさが強調されます。<br><br>この問題を回避するために、スムーズスキンのように、コントラストが大幅に変化しないピクセルを無視するようにフィルターに指示するしきい値が導入されました。 <br><br>先ほど示したジッパーのグラフィックで、ジッパーの横にあるテクスチャに注目してください。 しきい値が小さすぎてノイズを抑制できないため、画像ノイズが発生します。 |
| モノクロ | 画像の明るさ（強度）をアンシャープマスクする場合に選択します。<br><br>各カラーコンポーネントを個別にアンシャープマスクする場合は選択解除します。 |

[画像をシャープにする](sharpening-image.md#sharpening_an_image)も参照してください。

Adobe Dynamic MediaおよびImage Server[&#128279;](/help/using/assets/s7_sharpening_images.pdf)での画像のシャープ化も参照してください。

## 背景のノックアウト

ノックアウトの背景を使用すると、画像をアップロードするときに自動的に背景を削除できます。 このテクニックは、特定のオブジェクトへの注意を引き、煩雑な背景から際立たせるために便利です。

| ノックアウトの背景のオプション | 説明 |
| --- | --- |
| ノックアウトの背景 | ノックアウトの背景機能とオプションを有効または「オン」にします。 |
| 隅 | 必須。<br> ノックアウトする背景色の定義に使用する画像の角。<br>左上、左下、右上、または右下<b>から選択できます。</b> |
| Fill メソッド | 必須。 <br>設定したコーナーの位置からピクセルの透明度を制御します。<br>次の塗りつぶし方法から選択できます。<br>・ <b>塗りつぶし</b>：指定したコーナーに一致し、接続されているすべてのピクセルを透明にします。<br>・ <b> ピクセルの一致</b>：画像上の位置に関係なく、一致するすべてのピクセルを透明にします。 |
| 許容値 | オプション。<br>設定したコーナーの位置に基づいて、ピクセルカラーマッチングで許可されるバリエーションの量を制御します。<br> ピクセルのカラーを正確に一致させるには、値を0.0に設定します。 または、1.0の値を使用して、最大のバリエーションを作成します。 |
