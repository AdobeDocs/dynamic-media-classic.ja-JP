# スウォッチセットにズームターゲットと画像マップを含める{#including-zoom-targets-and-image-maps-in-swatch-sets}

スウォッチセット内の画像に対してズームターゲットと画像マップを定義すると、これらのターゲットと画像マップはスウォッチセットビューアに表示されます。 例えば、ユーザーはスウォッチセットビューアで画像マップを選択し、アイテムに関する情報を含むWeb サイト上の新しいページを開くことができます。 ユーザーは、スウォッチセットビューアでターゲットサムネール画像を選択し、画像の一部に自動的にズームできます。

ユーザーがズームターゲットと画像マップを使用できるようにするには、スウォッチセット内の画像のズームターゲットと画像マップを作成します。 これは、スウォッチセット画面または参照パネル（詳細表示）の画像マップまたはズームターゲットツールで行うことができます。
