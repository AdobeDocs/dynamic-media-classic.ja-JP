# 周辺光量補正、窓カバー、キャビネットファイルの操作{#working-with-vignette-window-covering-and-cabinet-files}

Adobe Dynamic Media Classicの画像オーサリングツールを使用しているユーザーは、*使用できる周辺光量補正（VNT ファイル）、ウィンドウのカバー（VNW ファイル）、およびキャビネット（VNC ファイル）を*&#x200B;使用できます。 これらのファイルは、Adobe Dynamic Media Classicで作成または編集できません。

周辺光量補正、窓カバー、およびキャビネットは、作成される画像です。 画像オーサリングと呼ばれる別のAdobe Dynamic Media Classic プログラムを使用して、画像内のオブジェクトをマスクし、同じ画像をさまざまなテクスチャやカラーで表示できるようにします。 たとえば、利用者が目にするのは、さまざまな布地で張られたソファ、さまざまな窓トリートメントで装飾された窓、さまざまな種類の木材で仕上げられたキャビネットです。

他のファイルと同様に、周辺光量補正、ウィンドウのカバー、キャビネットをアップロードします。
