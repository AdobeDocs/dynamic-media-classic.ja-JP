# Adobe Dynamic Media Classic ガイド {#scene7-overview}

Adobe Dynamic Media Classic ヘルプへようこそ。 利用可能なすべてのリソースにアクセスできるので、Adobe Dynamic Media Classic（旧Scene7）の使用について詳しく学ぶことができます。

## 2020年6月30日（PT）に新しいAdobe Dynamic Media Classic ログインが利用可能

2020年6月30日（PT）、Dynamic Media Classic ユーザーは、ブラウザーでAdobe Flash テクノロジに依存しなくなった新しいサインインにアクセスできるようになりました。

[よくある質問](new-ui-2020.md)を参照してください。 2020年5月7日時点の新しい更新。

## Dynamic Media開発者リソース

* [Dynamic Media開発者リソース ランディングページ &#x200B;](https://experienceleague.adobe.com/en/docs/dynamic-media-developer-resources)：次の内容が含まれます。
   * ビューアリファレンスガイド
   * 画像サービング/画像レンダリング API
   * Image Production System API
   * Adobe Dynamic Media Classic リリースノートを参照してください
* [&#x200B; ビューアライブラリの例](https://landing.adobe.com/en/na/dynamic-media/ctir-2755/live-demos.html)：ビデオ、ショッパブルメディア、混在メディア、スピンセット、インラインズーム、画像とスウォッチセット、およびフライアウトのビューアの例が含まれます。
