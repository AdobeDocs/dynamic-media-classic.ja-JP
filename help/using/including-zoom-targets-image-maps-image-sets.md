# 画像セットにズームターゲットと画像マップを含める{#including-zoom-targets-and-image-maps-in-image-sets}

画像セット内の画像に対してズームターゲットと画像マップを定義すると、これらのターゲットとマップは画像セットビューアに表示されます。 例えば、ユーザーは画像セットビューアで画像マップを選択し、アイテムに関する情報を含むWeb サイト上の新しいページを開くことができます。 ユーザーは、画像セットビューアでターゲットサムネール画像を選択し、画像の一部に自動的にズームできます。

ユーザーがズームターゲットと画像マップを利用できるようにするには、画像セット内の画像にズームターゲットと画像マップを作成します。 これには、画像セット画面または参照パネル（詳細表示）の画像マップまたはズームターゲットツールを使用します。
