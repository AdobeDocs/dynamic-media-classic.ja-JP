# Adobe Dynamic Media ClassicのMedia Portalの必要システム構成 {#system-requirements}

Adobe Dynamic Media Classic Media Portalを使用する最適なエクスペリエンスを得るには、お使いのシステムが次の必要システム構成と推奨事項を満たしていることを確認してください。

* Microsoft® Windows® 7以降またはmacOS X 10.6以降のMac Intel® プロセッサーが必要です。
* 3 GB以上のRAMが必要（4 GB以上を推奨）。
* カラーモニタ。
* 1280 x 1024以上の解像度でTrue Colorをサポートするビデオカード。
* Java対応ブラウザー：
* Windows でサポートされているブラウザー：Internet Explorer 9 以上、Firefox 23.0 以上。
* Macがサポートするブラウザー：Safari 6.0以降、Firefox™ 23以降。
