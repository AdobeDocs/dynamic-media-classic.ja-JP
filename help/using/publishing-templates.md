# テンプレートの公開{#publishing-templates}

テンプレートを公開すると、Web サイトとアプリケーションで使用できるDynamic Media画像サーバーに配置されます。 公開プロセス中に、Adobe Dynamic Media ClassicはWeb サイトとアプリケーションに必要なURLをアクティベートします。

>[!NOTE]
>
>テンプレートを使用するには、フォントや画像など、テンプレートを作成するのに使用したすべてのコンテンツを公開します。 必要なファイルをすべて含めないと、公開時にエラーメッセージが表示されます。

## テンプレートをパブリッシュ用にマーク {#marking-templates-for-publish}

テンプレートとそのすべてのサポートファイルをDynamic Media Image Serverに配置するには、公開するようにマークする必要があります。 これらの項目を参照パネルで公開用にマークするには、**[!UICONTROL 公開用にマーク]** アイコンを選択します。

## テンプレートを公開 {#publishing-your-template}

Dynamic Media画像サーバーにテンプレートを公開するには、グローバルナビゲーションバーで「**[!UICONTROL 公開]**」を選択し、「**[!UICONTROL 時]**」オプションを選択します。 必要に応じて、公開ジョブの名前を入力できます。 「**[!UICONTROL 公開を送信]**」を選択します。
