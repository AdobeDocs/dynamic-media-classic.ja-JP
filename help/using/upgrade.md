# [!DNL Dynamic Media]にアップグレードする理由

Adobeは、お客様を念頭に置いて[!DNL Dynamic Media]を作成しました。

[!DNL Adobe Experience Manager]の新しい[!DNL Dynamic Media]は、[!DNL Dynamic Media Classic] （旧Scene7）で期待されていた機能に基づいています。 これにより、より優れた魅力的なエクスペリエンスを構築するための、新しいツールや改善されたツールが手に入ります。

[!DNL Dynamic Media]は、リアルタイムのビジュアル ビルダーを使用して、コンテンツ構築エクスペリエンスをアップグレードします。 Adobeの信頼性と拡張性に優れた配信ネットワークのトップに位置し、直感的な[!DNL Adobe Experience Manager] ユーザーインターフェイスと組み合わせることで、[!DNL Dynamic Media]は魅力的で次のレベルのエクスペリエンスをオーサリング、反復、配信する機能を提供します。

## [!DNL Dynamic Media]の主な利点と機能

[!DNL Dynamic Media]は[!DNL Dynamic Media Classic]の自然進化です。 リッチメディアのオーサリング、公開、配信ツールセットを提供します。 また、革新的でユーザー中心のインターフェイスを備えたクラス最高の配信が含まれており、最新の現実世界のワークフローを促進します。

* シームレスで統合されたエクスペリエンスと[!DNL Adobe Experience Manager]との統合：1つのストア、1つのソースからリッチコンテンツを設定して動的に配信します
* リッチメディアコンテンツのオーサリングと配信を簡単にする、新しいマーケター中心のユーザーエクスペリエンス
* WYSIWYG（What You See Is What You Get）の全体的なルックアンドフィール、および画像ビューアとビデオプレーヤーのビヘイビアーエディターに関して。
* 画像のホットスポットやショッパブルビデオなど、ショッパブル/インタラクティブメディア体験の作成と展開を高速化
