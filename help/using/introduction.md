# Adobe Dynamic Media Classic ヘルプ {#scene7-overview}

Adobe Dynamic Media Classicは、業界をリードするオンデマンドのクロスメディアパブリッシングシステムです。 動的なリッチコンテンツを、あらゆるデバイスとスクリーンにアップロード、管理、強化、公開できます。 Adobe Dynamic Media Classicのソリューションには次のようなものがあります。

* Dynamic Imaging：リアルタイム編集、フォーマットとサイズ調整、インタラクティブなズームとパン、カラー`swatching`とテクスチャ `swatching`、360度スピン、画像テンプレート、マルチメディアビューア。
* e カタログ：オーサー、ホットスポットリンク、検索の有効化、公開、設定可能なビューアに動的に提供されるオンラインカタログの配信。
* Media Portal: ポータルユーザーがAdobe Dynamic Media Classic アセットに管理者が管理する「ビュー」を提供するセルフサービス環境により、アセットのアップロード、参照、検索、プレビュー、エクスポートを簡単に行うことができます。
* eVideo：任意の形式で最終的なビデオをアップロードし、管理、編集、ホットスポットとビデオテンプレートを作成し、設定可能なビデオビューアに公開してストリーミングします
* ビジュアルコンフィギュレーター：パーソナライゼーションとカスタマイズ効果のためのオンラインビジュアル設定を可能にするリアルタイムレンダリングテクノロジー。 カラー、テクスチャ、サーフェス、パターン、モノグラム、および製品画像やシーンへの彫刻のリアルタイム レンダリングなどのエフェクト。
* 対象者が絞り込まれた電子メールおよびプリント：顧客プロファイル、製品およびチャンネルデータベースとリンクし、視覚的に対象者を絞り込みパーソナライズした電子メールキャンペーンを配信します。
